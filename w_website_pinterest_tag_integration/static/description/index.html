Manage Pinterest Tags

Register odoo website actions in pinterest tags.

Features.
  1. Register all Webpages Visit event.
  2. Register user Signup event.
  3. Register user Add to Cart action in odoo shop webpage.
    Event Data:
    • Value: Order total value. (ex: 2500.00)
    • Order Quantity:Set order total product quantity.(ex: 3)
    • Currency: Order currency for payment. (ex: USD)
    • Order ID: Order identifier. (ex S00042)
    • Order Product List: (For each product)
      • Product Name: Order product name.(ex: Green Picture)
      • Product Category: Order product category.(ex: Decorations)
      • Product ID: Order odoo product identifier.(ex: 35)
      • Product Price: Order product price.(ex: 49.99)
      • Product Quantity: Order same product quantity.(ex: 2)
  4. Set user Checkout action in odoo shop webpage.
    Register the same Event Data like Add to Cart
Enable webpage config settings options to use Pinterest Tags.

To start to use this module, you have to configure the webpage configuration setting.

  • Select your Website if you manage more than one.
  • Scroll down to Features section.
  • Check Use Pinterest Tag options and type your website Pinterest Tag ID

Enjoy the Pinterest Tag Integration Module!
